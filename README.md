# pokemon-team-builder

A browser-based Pokémon team building tool that lets you search, compare and analyse a full team of six. Built as a personal project for use with friends and my partner — somewhere we could plan teams, share them with a link and get feedback on coverage and weaknesses without needing a dedicated app.

I have always been a fan of Pokémon and have used various team builders over the years, but I wanted to build one myself that felt clean, fast and personal. It started as a simple search card and grew from there as I kept adding features I actually wanted to use.

The project took around two months in total, though realistically around three and a half weeks of focused work to plan, design, code, link together, debug, test and deploy. I built it entirely in vanilla HTML, CSS and JavaScript with no frameworks, pulling live data from the PokéAPI. Features grew organically — the compare mode, evolution chains, type coverage analysis, team scoring and shareable URL links all came from sitting with the tool and thinking about what was genuinely missing.

It was also an opportunity to think carefully about how a project like this should be structured for public use — things like input sanitisation, accessible markup and clean event handling that I revisited and improved before publishing.

I hope it is useful to anyone who finds it and that it doesn't have too many bugs!
